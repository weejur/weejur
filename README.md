# weejur

**Publish websites for free with GitHub Pages**

Turn your HTML into a live website in seconds - just paste HTML or upload files. Completely free.

**[weejur.com](https://weejur.com)**

## How it works

1. Paste HTML or upload files
2. Sign in with GitHub
3. Your site gets a public URL in seconds

Works great with code from ChatGPT, Claude, and other AI tools.
